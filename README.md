In its basic form it is a copy/sync tool, in that it copies files from source folder to a destination folder. Rsync backs up files using the native file system of your computer. It does not have its own compressed or proprietary database. You can easily use your </span><span class="c3">File Manager</span><span>&nbsp;to restore with drag and drop any backup folder or individual files.
